ECHO, the executable CHOndrocyte: A computational model to study articular chondrocytes in health and disease

Stefano Schivo, Jetse Scholma, Johan Kerkhofs, Leilei Zhong, Kannan Govindaraj, Xiaobin Huang, Jaco C. van de Pol, Rom Langerak, Andre J. van Wijnen, Liesbet Geris, Marcel Karperien, Janine N. Post


ECHO_AC_Model_Runx2.cys: ECHO model, articular chondrocyte (AC) version (Model 4), RUNX2-positive initial state
ECHO_AC_Model_Sox9.cys: ECHO model, articular chondrocyte (AC) version (Model 4), SOX9-positive initial state

ECHO_parameters.docx: the parameters of the ECHO models (both growth plate and articular chondrocyte version):
  - node activities in the RUNX2-positive and SOX9-positive state
  - k-parameters for the interactions

Scripts.zip: all the scripts and additional models used to generate the figures in the paper. This file contains an additional readme file with an index of the experiments.
